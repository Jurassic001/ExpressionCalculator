# ExpressionCalculator

Iterates through an equation and pushes numbers and operations onto two different ArrayLists that act as Stacks.

Program accepts multiple digits in a number, and the operations +, -, *, /, and ^

#### Examples:
Input:
5+5-5 <br/>
Output:
5

Input:
4/2*3^2<br/>
Output:
18

Input:
2^3/2^3<br/>
Output:
1

Input:
1+3*4/3-2^3/2<br/>
Output:
1

Input:
1+1+1+1+1+1+2^3<br/>
Output:
14

Input:
1-1+1-1+3/3*3<br/>
Output:
3

Input:
3*4^2+5<br/>
Output:
53

Input:
23+2<br/>
Output:
25
